# Memory Game

This is a memory game project as part of The Odin Project's React course curriculum. This project is meant to showcase my understanding of: 
   - The useEffect React Hook
   - Predictable state management and updates
   - Avoiding infinite re-renders

The languages I used for this project are: 
   - React
   - CSS

Features: 
   - Shuffling card layout on state change
   - Tracking clicked cards with Set
   - Score tracker that increases when cards are clicked
     and resets when the same card is clicked twice
